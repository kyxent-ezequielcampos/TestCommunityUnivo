# TestCommunity - Fullstack CRUD App

This repository contains a complete **Products + Providers** management system with:

- ✅ Backend REST API (Bun + Elysia + MongoDB)
- ✅ Frontend UI (React + Vite + Tailwind/DaisyUI)
- ✅ Full CRUD + relationships + filtering/sorting/pagination
- ✅ Dockerfiles for easy deployment (Dokku, Docker, etc.)

---

## 🧱 Structure
- `/backend` - REST API server
- `/frontend` - React UI

---

## 🚀 Quick start
### Backend
```bash
cd backend
bun install
bun run dev
```

### Frontend
```bash
cd frontend
bun install
bun run dev
```

---

## 📌 Deployment (Docker / Dokku)
Each side has a Dockerfile at its root. Build and run from each folder.
